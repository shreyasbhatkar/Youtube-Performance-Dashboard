# Youtube-Performance-Dashboard
This project is an interactive Tableau dashboard analyzing global YouTube channel performance, including subscribers, views, earnings, and category trends.

Tools Used:
Tableau
Excel / CSV

Key Insights:
Identified top-performing channels by subscribers
Compared category-wise performance
Analyzed correlation between views and earnings
Visualized global distribution of creators

Files Included:
Tableau Workbook (.twbx)
Dataset (.csv)
